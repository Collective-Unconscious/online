# Collective Unconscious Online

This is the repository for the Web frontend for Collective Unconscious. There
has been a motion among developers to move into SvelteKit instead of React for
development of the website, and development of the front- and backend were both
in early planning stages when this transpired, so starting over from scratch
was more suitable than trying to copy over what was simply a barebones Discord
login page developed by someone who ended up swiftly leaving the project after
starting development.

I'm setting this up rather hastily but expect some proper information to come
about in this document very soon. Main development takes place on the `main`
branch. Please use pull requests and separate branches/forks to work on your
changes.
